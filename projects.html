MoodBloom

Tech stack: React | Node | Apollo | GraphQL | MongoDB | Passport.js


For this project I worked in a cross-functional team, building a mood-tracking progressive web app (PWA). The app allows users to signup (locally or with Google, if desired), and then presents them with a dashboard where they can record their mood, amount of sleep, anxiety level, and other notes. Users can complete activities, view a calendar of their mood history, or export their data as a PDF.

DadJoke Project

Tech stack: Java | Spring | Spring Boot | PostgreSQL | React | Redux


For this project I created a Java/Spring backend API with CRUD functions for a frontend. The API has private/public modes, allows users to signup and login, and accepts query strings where applicable. I later built a react frontend to showcase the project.

Action-Adventure Game

Tech stack: C# | Unity


Built from scratch in Unity, I do the coding while my co-creator makes the art assets. Implemented a multi-slot save system using a singleton class to save current state values, and then write them to binary. Coded complex game mechanics, including a programatically-generated quest log, bestiary, and lore compendium using static classes. Implemented walk and dash, weapons, enemies, sub-weapons (bow and error, fire rod, magic, etc.), and NPCs.

Code is stored in a private repository, available on request.

SwitchList

Tech stack: React | Redux | Node | REST API | MongoDB


This project is a list of Nintendo Switch games, drawn from a Node.js API which I also constructed. The site allows people to access a list of games and sort by many options, including publisher, developer, composer, genre, and more. The project is built using React and React-Router, while the state is managed with Redux.
