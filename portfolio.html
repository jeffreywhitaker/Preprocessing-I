MoodBloom

Tech stack: React, Apollo, GraphQL, MongoDB, Passport.js


For this project I worked in a cross-functional team, building a mood-tracking progressive web app (PWA). The app allows users to signup (locally or with Google, if desired), and then presents them with a dashboard where they can record their mood, amount of sleep, anxiety level, and other notes. Users can complete activities, view a calendar of their mood history, or export their data as a PDF.

DadJoke Project

Tech stack: Java, Spring, Spring Boot, SQL, React, Redux


For this project I created a Java/Spring backend API with CRUD functions for a frontend. The API has private/public modes, allows users to signup and login, and accepts query strings where applicable. I later built a react frontend to showcase the project.

SwitchList

Tech stack: React, Redux, Node.js


This project is a list of Nintendo Switch games, drawn from a Node.js API which I also constructed. The site allows people to access a list of games and sort by many options, including publisher, developer, composer, genre, and more. Users are able to create an account and, once they are logged in, can sort games into private containers to indicate, for example, which ones they already own, want, or like. The project is built using React and React-Router, while the state is managed with Redux.
